Please note:
Most of the items listed are either from Hololive or things I just like.
This website is a demo & cannot be used to purchase anything.


However, all the functionality you would find in a typical e-commerce website is here (Except for actually delivering your order). The website itself was made using the .NET framework with Angular. PostgreSQL is used for the database, and the website is hosted using Digital Ocean.

I did not want to buy a domain name for just a demo website. I am not made of money.


Here are some features of Holonet:

  • Filter and search through a paginated database of things I like
  • Create a basket of products you want, which will persist with your account
  • Register and login with an account to keep track of your orders
  • APIs with DTOs for easy information sharing
  • Responsive interface with validation on all forms
  • Stripe API integration for secure checkout *
